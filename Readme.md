# 🕹️ Jogo do Número Secreto - Logic Challenge

### 📋 Visão Geral
Este projeto consiste em um jogo interativo de adivinhação desenvolvido com tecnologias web fundamentais. O objetivo é demonstrar o domínio de lógica de programação, manipulação de estados (tentativas) e a integração entre lógica JavaScript e interface visual.

---

### 🚀 Funcionalidades Principais
* **Lógica de Adivinhação:** O sistema processa um número secreto (atualmente definido como 17) e valida os inputs do usuário em tempo real.
* **Feedback Inteligente:** O código fornece dicas se o número secreto é maior ou menor que o chute atual do jogador.
* **Contador de Performance:** O jogo rastreia e exibe a quantidade de tentativas necessárias para a vitória.
* **Interface Responsiva:** Layout estruturado com HTML5 semântico e CSS moderno para centralização e profundidade visual.

---

### 🛠️ Tecnologias Utilizadas
* **JavaScript (ES6+):** Utilizado para o controle de fluxo com estruturas `while`, condicionais `if/else` e manipulação de variáveis de estado.
* **HTML5:** Estruturação da página com integração de scripts assíncronos via atributo `defer`.
* **CSS3:** Estilização avançada com uso de `linear-gradient`, Flexbox para alinhamento e fontes externas da biblioteca Google Fonts ('Chakra Petch' e 'Inter').

---

### 🧠 Lógica de Desenvolvimento
Como gestor de sistemas, destaco os seguintes pontos técnicos deste repositório:
1.  **Iteração Dinâmica:** O uso de um loop `while` garante que a experiência do usuário seja contínua até a resolução do desafio proposto no jogo.
2.  **User Experience (UX):** A interface utiliza métodos de entrada e saída direta (`prompt` e `alert`) para manter o fluxo de interação simples e funcional.
3.  **Estética Profissional:** O design utiliza camadas de fundo (`background-image` e `opacity`) para criar uma identidade visual moderna no container principal.

---

### 📂 Como Rodar o Projeto
1.  Clone este repositório.
2.  Abra o arquivo `index.html` em qualquer navegador moderno.
3.  Tente adivinhar o número entre 1 e 30.
